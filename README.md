# utp2017-5-excel
Браузерная электронная таблица

**Возможности:**
 * Изменение размеров таблицы
 * Редактирование содержимого ячеек. Смена цвета, стиля, размера шрифта
 * Поддержка формул
 * Обработка рекурсивных зависимостей в формулах
